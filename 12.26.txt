리눅스 단축키 정리
$cd -> 뒤의 경로로 이동
$cd ~ -> home 경로로 이동
$cd .. ->상위 경로로 이동
$pwd -> 현재 디렉토리 확인
$ls -> 현재 디렉토리의 파일 목록 보여줌
$ls -al -> 숨김 파일 포함하여 보여줌
$cal -> 달력
$date -> 날짜
$who -> 로그인 정보
$dpkg -> 설치된 패키지 정보
$echo -> home 경로
$cat -> 파일을 보여줌
$nl -> 번호를 포함하여 파일을 보여줌
$head -> 위에서부터 10개줄 출력
$tail -> 밑에서부터 10기줄 출력

상대경로: /부터 시작
절대경로: pwd가 기준이됨
~ -> home 경로로 이동
/ -> 최상위 경로로 이동

touch -> 파일생성
mkdir -> 폴더생성
cp -> 복사
mv -> 이동
rename -> 이름변경
rm -> 파일 제거
rmdir -> 폴더 제거

-f -> 강제삭제
